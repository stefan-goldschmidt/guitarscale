# Guitar Scale Visualizer

A modern, interactive tool for guitarists to visualize scales and the Circle of Fifths.

## Features

- **Interactive Circle of Fifths**: Toggle between Major and Minor keys with a single click.
- **Dynamic Fretboard**: Instantly see scales mapped across the entire fretboard.
- **Musical Theory Logic**: Correct mapping of parallel keys (e.g., A Minor shows the same notes as C Major).
- **Modern UI**: Clean, "Glassmorphism" inspired design with a focus on readability.
- **Responsive**: Optimized for both desktop and mobile devices.

## Vibecoding Experiment

This project is a **Vibecoding** experiment. It was created to test and compare the capabilities of various Large Language Models (LLMs) in a real-world development scenario. The entire codebase, design, and logic were generated and refined through iterative prompting.
